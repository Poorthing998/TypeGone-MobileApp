# iOS Custom Keyboard Setup

Since you are using React Native (Expo bare workflow), the iOS Keyboard Extension requires adding a new Target to the Xcode project manually. Here is how to do it:

1. Open `ios/TypeGoneMobile.xcworkspace` in Xcode.
2. Go to **File -> New -> Target...**
3. Select **Custom Keyboard Extension** and click Next.
4. Name the target `TypeGoneKeyboard`. Make sure the language is **Swift**. Click Finish.
5. Xcode will ask to activate the scheme. Click **Activate**.
6. Replace the autogenerated `KeyboardViewController.swift` and `Info.plist` inside the newly created `TypeGoneKeyboard` folder in Xcode with the versions provided in this directory.
7. Under the `TypeGoneKeyboard` target settings, ensure that **App Groups** capability is added (and the same App Group is added to your main app target) so that they can share UserDefaults (API keys, preferences).

You can then run the app on an iOS Simulator or Device, go to Settings -> General -> Keyboard -> Keyboards -> Add New Keyboard, and select `TypeGoneMobile`.
